# Mennekes AMTRON Wallbox – Home Assistant Integration

Complete integration of a Mennekes AMTRON Wallbox in Home Assistant.

## Installation

### HACS
1. HACS → Automations
2. Search "Mennekes AMTRON"
3. Install

## Quick Start

1. Copy `.env.example` to `.env`
2. Update `WALLBOX_URL` and `WALLBOX_PASS`
3. Copy files to `/config/`
4. Restart Home Assistant

## Features

- Real-time Modbus TCP monitoring
- Charging sessions via REST API
- RFID vehicle identification
- Cost calculation
- ApexCharts dashboard
